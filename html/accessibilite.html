Le Scribouill'art s'engage à rendre son contenu accessible à tous les utilisateurs, quelles que soient leurs capacités ou les technologies qu'ils utilisent.

Accessibilité

Notre engagement

Le Scribouill'art s'efforce de respecter les standards d'accessibilité du web (WCAG 2.1 niveau A/AA) afin de garantir une expérience de navigation inclusive pour tous les visiteurs.

Fonctionnalités d'accessibilité

Mode dyslexie

Un bouton en haut à gauche de chaque page permet d'activer une police spécialement conçue pour faciliter la lecture des personnes dyslexiques (OpenDyslexic). Ce mode est sauvegardé dans votre navigateur.

Navigation au clavier

Toutes les fonctionnalités du site sont accessibles au clavier :

  • Tab : Naviguer entre les éléments interactifs
  • Shift + Tab : Naviguer en arrière
  • Entrée ou Espace : Activer un lien ou un bouton
  • Lien "Aller au contenu principal" : Apparaît en haut de page lors de la première pression sur Tab, permettant de sauter directement au contenu principal

Structure sémantique

Le site utilise des balises HTML sémantiques appropriées (header, nav, main, footer, article) et une hiérarchie de titres cohérente pour faciliter la navigation avec les lecteurs d'écran.

Alternatives textuelles

Toutes les images importantes disposent de descriptions alternatives (attribut alt) permettant aux utilisateurs de lecteurs d'écran de comprendre leur contenu.

Indicateurs de focus visibles

Les éléments interactifs (liens, boutons, champs de formulaire) affichent un contour visible lorsqu'ils reçoivent le focus au clavier.

Technologies compatibles

Le site est compatible avec les principaux lecteurs d'écran :

  • NVDA (Windows)
  • JAWS (Windows)
  • VoiceOver (macOS, iOS)
  • TalkBack (Android)

Navigateurs recommandés : Chrome, Firefox, Safari, Edge (versions récentes).

Niveau de conformité

Le Scribouill'art vise une conformité au niveau A des WCAG 2.1 (Web Content Accessibility Guidelines). Nous travaillons continuellement à améliorer l'accessibilité de notre site.

Signaler un problème d'accessibilité

Si vous rencontrez des difficultés pour accéder à certains contenus ou fonctionnalités du site, n'hésitez pas à nous le signaler via notre page de contact. Nous nous efforcerons d'apporter les corrections nécessaires dans les meilleurs délais.
